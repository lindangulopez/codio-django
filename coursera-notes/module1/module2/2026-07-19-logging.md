### What this means (beginner explanation)

Your script created a logger:

```python
logger = logging.getLogger(__name__)
```

Because you ran:

```bash
python3 logging_example/logging_practice.py
```

Python sets:

```python
__name__ = "__main__"
```

So your logger appears as:

```text
DEBUG:__main__
INFO:__main__
WARNING:__main__
```

The format is:

```text
LEVEL : LOGGER NAME : MESSAGE
```

For example:

```text
ERROR:__main__:This is an error message
```

means:

* **ERROR** → the importance level
* ****main**** → the file that created the log
* **This is an error message** → the actual message
